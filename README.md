После запуска app.rb приложение будет доступно по ссылке: http://localhost:4567/
Приложение на Sinatra - тест на знание английских слов.
Генерирует тест из 10-ти случайных слов из базы. Проверяет ответы и выдаёт результат.
Слова, на которые 10 раз был получен правильный ответ, считаются выученными и исключаются из тестов.
